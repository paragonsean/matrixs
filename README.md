# seanspnmatrix
a library for solving linear equations based on c++17.

support:

* Gmres-m.

* Jacobi iteration.

* Gauss-seidel iteration.

```
mkdir build
cd build
cmake ..
make
```
Then you can find executables in folder build/bin. Try to execute them:)
```
cd bin
./pnmatrix_example
./pnmatrix_test
```
you can find source code in folder examples and test
